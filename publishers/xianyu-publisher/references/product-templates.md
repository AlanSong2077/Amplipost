# 闲鱼商品模板

## 3C数码类

### 手机模板

**标题：**
```
【{新旧程度}】{品牌} {型号} {内存} {颜色} {版本} {卖点} {价格}
```

**描述：**
```
📱{品牌}{型号} {内存}+{存储}

【配置】{详细配置}
【新旧程度】{新旧描述}
【颜色】{颜色}
【电池】{电池健康度}
【配件】{配件情况}
【入手渠道】{购买渠道}
【出手原因】{原因}

📦【配件】{配件清单}
🚝【交易】{交易方式}
💬【说明】{其他说明}

有意请留言，看到会第一时间回复～
```

### 电脑模板

**标题：**
```
{品牌}{型号} {芯片} {内存}+{硬盘} {新旧程度} {卖点} {价格}
```

**描述：**
```
💻{品牌}{型号}

【配置】
- 芯片：{芯片型号}
- 内存：{内存大小}
- 存储：{硬盘大小}
- 屏幕：{屏幕尺寸}
- 电池：{电池循环次数/健康度}

【新旧程度】：{新旧描述}
【外观】：{外观描述}
【配件】：{配件清单}
【购买渠道】：{购买时间/渠道}
【出手原因】：{原因}

✅{检查项目清单}
🎁{赠品}

📦{交易方式}
🔔{其他说明}
```

## 通用模板

如果不确定分类，可使用通用模板：

**标题：**
```
【{新旧程度}】{商品名称} {核心卖点} {价格}
```

**描述：**
```
{商品名称}

【商品情况】：{详细介绍}
【新旧程度】：{新旧}
【入手时间】：{时间}
【出手原因】：{原因}

📦{交易方式}
💬{其他说明}

诚心要可议价，谢谢！
```
